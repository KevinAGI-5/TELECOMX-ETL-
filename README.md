# TELECOMX-ETL 📊📱

Este repositorio contiene un pipeline de **ETL (Extract, Transform, Load)** desarrollado en un entorno de Jupyter Notebook (`telecomx1.ipynb`). El proyecto está enfocado en el procesamiento de datos del sector de telecomunicaciones, permitiendo limpiar, analizar y preparar grandes volúmenes de información para la toma de decisiones.

## 🚀 Descripción del Proyecto

El objetivo de este proyecto es simular o ejecutar un flujo de trabajo de ingeniería de datos donde se transforman datos brutos de telecomunicaciones (como registros de llamadas, uso de datos o perfiles de clientes) en un formato estructurado y optimizado.

### Fases del Pipeline:
1.  **Extracción (Extract):** Lectura de fuentes de datos (CSV, JSON o SQL).
2.  **Transformación (Transform):** - Limpieza de valores nulos y duplicados.
    - Normalización de formatos de fecha y tipos de datos.
    - Creación de nuevas métricas (Feature Engineering).
3.  **Carga (Load):** Exportación de los datos procesados a un nuevo archivo o base de datos para su posterior análisis.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Librerías principales:**
    * `Pandas`: Manipulación y análisis de datos.
    * `NumPy`: Operaciones numéricas.
    * `Matplotlib` / `Seaborn`: Visualización de datos (si aplica).
    * `Jupyter Notebook`: Entorno de desarrollo interactivo.

## 📂 Estructura del Repositorio

* `telecomx1.ipynb`: Notebook principal que contiene todo el código del proceso ETL, documentado paso a paso.
* `data/`: (Opcional) Carpeta sugerida para almacenar los datasets de entrada y salida.
